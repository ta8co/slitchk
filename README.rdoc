== すりっとちぇっかー（仮）

=== 1.Description

出勤・退勤時のカードリーダーの通し忘れがないかどうかをチェックする為のアプリです。

=== 2.Getting Started

(1) clone this repogitory

(2) bundle install

(3) rake db:migrate

(4) (optional)rake db:seed

"seeds.rb" inculde sample data.
If you want to use original data, access (your_url)/members and create data.
